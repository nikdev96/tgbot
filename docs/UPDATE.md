# 📋 История обновлений Translation Bot

## 📷 v2.7.5 - Photo OCR + Translation (Март 2026)

### ✅ Завершённые задачи

#### 📷 Поддержка фотографий
- **OCR через GPT-4o Vision** - Извлечение текста со скриншотов, вывесок и документов
- **Интеграция с пайплайном перевода** - Извлечённый текст передаётся в существующий `process_translation()`
- **Предпросмотр текста** - Бот показывает найденный текст перед переводом: `🔤 Текст: ...`
- **Лимит размера** - Файлы до 20 МБ
- **Retry логика** - 3 попытки с экспоненциальным backoff, таймаут 45 сек
- **Динамическая модель** - Использует текущую модель из `model_manager` (gpt-4o/gpt-4o-mini)

#### 🧹 Очистка проекта
- Удалены backup-файлы (`keyboards.py.backup`, `voice.py.backup`)
- Удалены одноразовые скрипты (`migrate_ko_to_zh.py`)
- Удалены тестовые файлы из корня (`test_room_commands.py`, `test_room_integration.py`)
- Удалены устаревшие planning-документы из `docs/`
- Исправлены несоответствия в документации (6 языков → 4 языка: ru, en, th, vi)

#### 📁 Новые файлы
- `src/services/vision.py` - GPT-4o Vision OCR сервис
- `src/handlers/photo.py` - Обработчик фото-сообщений

## 🎯 v2.6.0 - Переговорные комнаты + Умные настройки (Октябрь 2024)

### ✅ Завершенные задачи

#### 🏠 Переговорные комнаты для многостороннего общения
- **Создание комнат** - FSM мастер с выбором названия и языка
- **Присоединение по коду** - `/room join CODE` с выбором своего языка
- **Автоперевод в реальном времени** - Каждый участник пишет на своём языке, получает перевод
- **Управление комнатами** - Просмотр участников, выход из комнаты, закрытие (для создателя)
- **Система приглашений** - Простой шаринг с кодом доступа
- **Поддержка голоса** - Голосовые сообщения транскрибируются и переводятся в комнатах
- **Валидация** - Проверка заполненности, истечения срока, прав доступа
- **Уведомления** - Оповещения участников при входе/выходе

#### 🗄️ База данных для комнат
- **Таблица rooms** - Хранение комнат с уникальным кодом, названием, создателем
- **Таблица room_members** - Участники с языковыми предпочтениями для комнаты
- **Таблица room_messages** - История сообщений комнаты
- **Индексы** - Оптимизация запросов по кодам комнат и user_id
- **Автоочистка** - Удаление истёкших комнат при старте

#### 🔧 Умная фильтрация языков перевода
- **Уважение предпочтений** - Переводит только на выбранные пользователем языки
- **Фикс дефолтного поведения** - Если язык сообщения = единственный выбранный → показывает подсказку
- **Предотвращение перевода на все языки** - Больше не переводит на все языки при пустых предпочтениях после фильтрации

#### 🎨 UI/UX улучшения
- **Inline клавиатуры** - Удобная навигация по меню комнат
- **Эмодзи флаги** - Визуальное отображение языков участников
- **Статус участников** - Отображение создателя (👑) и обычных участников (👤)
- **FSM навигация** - Пошаговое создание и присоединение к комнатам
- **Алерты** - Информативные всплывающие подсказки с `show_alert=True`

#### 🐛 Исправленные проблемы
- **Фикс дублирования ответов** - Добавлено debug логирование для отслеживания вызовов
- **Правильная обработка голоса в комнатах** - Транскрипция → обнаружение языка → перевод → рассылка
- **Корректная очистка FSM** - Правильная очистка состояния при отмене операций
- **Умная логика перевода** - Теперь переводит только на выбранные языки, не на все

#### 📚 Документация
- **Обновлён README.md** - Добавлена информация о переговорных комнатах
- **Команды** - `/room` и `/room join CODE` задокументированы
- **UPDATE.md** - Полный changelog v2.6.0

## 🚀 v2.5.0 - Админ-панель + Оптимизация производительности (Октябрь 2024)

### ✅ Завершенные задачи

#### ⚡ Оптимизация обработки голосовых сообщений
- **Early Response** - Транскрипция показывается мгновенно (~3с), перевод обрабатывается в фоне
- **Устранение задержек** - Убраны ненужные обновления статусных сообщений между этапами
- **Улучшенная производительность** - 8-9с общее время (было 11.7с), ощущается как в 2 раза быстрее
- **Параллельная TTS генерация** - Все языки обрабатываются одновременно

#### 🔧 Рефакторинг админ-панели
- **Разделённое меню** - 3 отдельных раздела: Пользователи, Сервер, Очистка
- **Улучшенное управление пользователями** - Сортировка по последней активности
- **Мониторинг сервера** - CPU, память, диск, uptime, статус службы
- **Меню очистки** - Статистика и управление данными
- **Фикс ошибок** - Обработка "message is not modified" при повторных нажатиях

#### 🧹 Система автоматической очистки
- **Удаление неактивных пользователей** - Автоматическое удаление пользователей неактивных >3 дней
- **Очистка TTS кэша** - Удаление старых голосовых файлов >3 дней
- **Очистка при старте** - Автоматическая очистка при каждом запуске бота
- **Ручное управление** - Кнопки в админ-панели для ручной очистки
- **Статистика** - Отображение количества неактивных пользователей и размера кэша

#### 🗄️ Улучшения базы данных
- **Методы очистки** - `delete_inactive_users()` и `clear_tts_cache()`
- **Сортировка по активности** - Пользователи сортируются по `last_activity`
- **Защита данных** - Корректное удаление с учётом foreign key constraints
- **Логирование** - Подробное логирование операций очистки
