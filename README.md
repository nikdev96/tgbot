# 🌍 Telegram Переводчик

[![Python](https://img.shields.io/badge/Python-3.11+-blue.svg)](https://www.python.org/downloads/)
[![aiogram](https://img.shields.io/badge/aiogram-3.x-green.svg)](https://docs.aiogram.dev/)
[![OpenAI](https://img.shields.io/badge/OpenAI-GPT--4o-orange.svg)](https://openai.com/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Умный Telegram бот для перевода текста, голосовых сообщений и фотографий между **4 языками** 🇷🇺🇺🇸🇹🇭🇻🇳 с настраиваемыми пользовательскими предпочтениями и групповыми комнатами для многостороннего общения.

## 🛠️ Tech Stack

### Core Technologies
- **Python 3.13** - Основной язык программирования
- **aiogram 3.x** - Асинхронный фреймворк для Telegram Bot API
- **SQLite** - Встроенная база данных для хранения пользователей и настроек
- **FFmpeg** - Обработка аудио (конвертация и обработка голосовых сообщений)

### AI & ML
- **OpenAI GPT-4o/GPT-4o-mini** - Перевод текста с контекстом и эмодзи
- **OpenAI GPT-4o Vision** - Извлечение текста из фотографий (OCR)
- **OpenAI Whisper** - Распознавание голосовых сообщений (speech-to-text)
- **OpenAI TTS** - Синтез голоса для ответов (text-to-speech)
### Libraries & Tools
- **aiohttp** - Асинхронные HTTP запросы
- **cachetools** - In-memory кэширование переводов и TTS
- **langdetect** - Автоопределение языка текста
- **PyYAML** - Конфигурация через YAML файлы
- **pytest** - Тестирование с моками

### Infrastructure
- **systemd** - Управление сервисом в production
- **journalctl** - Логирование и мониторинг
- **Git** - Контроль версий

## ✨ Возможности

### 📷 Фотографии
- **OCR через GPT-4o Vision** - Извлечение текста со скриншотов, вывесок, документов
- **Автоматический перевод** - Извлечённый текст сразу передаётся в пайплайн перевода
- **Предпросмотр** - Бот показывает найденный текст перед переводом

### 🔄 Переводы
- **Автоопределение языка** - Русский, английский, тайский, вьетнамский
- **Настройки пользователя** - Выбор целевых языков через `/menu`
- **Умная фильтрация** - Переводит только на выбранные языки (не на все)
- **Голосовые сообщения** - Распознавание через Whisper + перевод
- **Голосовые ответы** - TTS с переключением для каждого пользователя
- **Сохранение эмодзи** - Эмодзи корректно передаются через все языки 😊🎉
- **Надёжный парсинг** - Marker-формат `[XX]...[/XX]` вместо JSON — устойчив к нестандартным ответам
- **Без цитирования** - Переводы приходят как самостоятельные сообщения, не цитируя оригинал
- **Без префиксов** - Чистые переводы без языковых меток

### 🏠 Переговорные комнаты
- **Создание комнат** - Удобный мастер с выбором названия и языка
- **Присоединение** - Команда `/room join CODE` с выбором своего языка
- **Автоперевод** - Каждый пишет на своём языке, все получают перевод
- **Управление** - Просмотр участников, выход, закрытие комнаты (для создателя)
- **Приглашения** - Простая система шаринга с кодом доступа
- **Поддержка голоса** - Голосовые сообщения работают в комнатах

### ⚡ Производительность и надёжность
- **Early Response** - Транскрипция показывается мгновенно (~3с), перевод в фоне
- **Параллельная обработка** - TTS генерация для всех языков одновременно
- **Умное кэширование** - Переводы (1ч) + TTS (30мин) с постоянным хранением
- **Защита от гонок** - Атомарные операции в базе данных
- **Автоочистка** - Удаление старых пользователей (>7 дней) и кэша при старте
- **Валидация переводов** - Проверка полноты переводов на все запрошенные языки
- **Fallback механизмы** - Отправка оригинального текста если перевод недоступен
- **Детальное логирование** - Отслеживание успешности переводов и отсутствующих языков

### 👑 Админ панель
- **Разделённое меню** - Отдельные разделы для пользователей, сервера и очистки
- **Управление пользователями** - Включение/отключение с сортировкой по активности
- **Выбор AI модели** - Динамическое переключение между GPT-4o и GPT-4o-mini
- **Мониторинг сервера** - CPU, память, диск, статус службы с кнопками 🔄 Refresh и 🔙 Back
- **Очистка данных** - Удаление неактивных пользователей и старого TTS кэша
- **Аналитика** - Счётчики неактивных пользователей, размер кэша

## 🛠️ Установка

```bash
git clone https://github.com/yourusername/tgbot
cd tgbot
python3 -m venv .venv
source .venv/bin/activate  # Linux/Mac
pip install -r requirements.txt
```

## ⚙️ Настройка

1. **Создайте `.env` файл:**
```bash
cp .env.example .env
```

2. **Заполните переменные:**
```bash
TELEGRAM_BOT_TOKEN=your_bot_token
OPENAI_API_KEY=sk-your-openai-key
```

3. **Настройте конфигурацию:**
- `config/development.yaml` - для разработки
- `config/production.yaml` - для продакшена

## 🚀 Запуск

### Разработка
```bash
source .venv/bin/activate
python -m src.main
```

### Продакшен (systemd)
```bash
sudo cp translator-bot.service /etc/systemd/system/
sudo systemctl daemon-reload
sudo systemctl enable translator-bot
sudo systemctl start translator-bot
```

## 📝 Команды бота

- `/start` - Начать работу с ботом
- `/menu` - Настройки языков перевода
- `/room` - Создать или присоединиться к переговорной комнате
- `/room join CODE` - Присоединиться к комнате по коду
- `/admin` - Админ панель (только для админов)
- `/help` - Справка по командам

## 🐛 Решение проблем

### Python 3.13 Совместимость
Бот полностью совместим с Python 3.13 и использует ffmpeg для обработки аудио (без зависимости от устаревшего модуля `audioop`).

### Неполные переводы
**v2.7.3** включает улучшения переводов:
- Marker-формат `[XX]...[/XX]` вместо JSON — устойчив к любым ответам модели
- Стриминг переводов: каждый язык появляется по мере готовности
- Переводы не цитируют оригинальное сообщение (чище в групповых чатах)
- Валидация полноты переводов и fallback для отсутствующих языков
- Корректная обработка эмодзи
- Модель выбирается через `OPENAI_MODEL` env var

### Нехватка места на диске
```bash
# Очистка логов
sudo journalctl --vacuum-time=7d
rm -rf logs/archive/*
```

### Проблемы с FFmpeg
```bash
# Ubuntu/Debian
sudo apt update && sudo apt install ffmpeg

# Проверка
ffmpeg -version
```
